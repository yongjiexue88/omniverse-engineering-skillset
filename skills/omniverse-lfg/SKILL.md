# Omniverse LFG

Use this skill when momentum matters but engineering guardrails still apply.

## Workflow

1. Clarify only blockers that cannot be discovered locally.
2. Inspect the repo fast: status, package scripts, key files, existing patterns.
3. Choose the smallest useful path and start.
4. Keep a short running checklist for multi-step work.
5. Validate before declaring done.
6. Leave a clear handoff: what changed, what passed, what remains.

## Rules

- Bias toward action, not extended planning.
- Do not bypass safety: secrets, destructive git commands, production state, and unrelated user work remain protected.
- If the task becomes larger than expected, pause long enough to reset scope and continue.
