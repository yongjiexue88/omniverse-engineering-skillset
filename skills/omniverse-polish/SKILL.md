# Omniverse Polish

Polish the experience without redesigning the product from scratch.

## Workflow

1. Identify the critical workflow or screen.
2. Check layout, hierarchy, spacing, copy, responsive behavior, and state coverage.
3. Verify error, empty, loading, disabled, and success states.
4. Fix issues that change comprehension or trust.
5. Avoid cosmetic churn that does not improve use.

## Checklist

- Text fits its container.
- Buttons and controls use familiar affordances.
- Primary action is clear.
- Errors tell users what happened and what to do.
- Loading states do not cause layout jumps.
- Keyboard and screen-reader basics are not broken.

## Rules

- Make the product easier to use, not more decorative.
- Match existing design language.
- Verify visually when a browser or screenshot path is available.
