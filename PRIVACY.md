# Privacy

Omniverse Engineering Skillset is a local skill package. It does not run a hosted service and does not intentionally collect telemetry.

## Local Data

The skills may instruct an agent to read or write local project files such as code, docs, plans, QA reports, and solution notes. Those files stay in the user's local repository unless the user or agent commits, pushes, uploads, or shares them through another tool.

## External Services

Some workflows mention optional tools such as GitHub CLI, browser automation, or Xcode tooling. Those tools may have their own authentication, logging, and data-handling behavior. Review those tools separately before using them with private repositories or sensitive data.

## Sensitive Information

Do not put secrets, credentials, private customer data, or sensitive logs into generated plan, proof, QA, or solution-note artifacts.
