# Omniverse Engineering Skillset

Use this skill when you want a coding agent to review non-trivial engineering work before it ships.

## Core Uses

| Use case | What the skill contributes |
| --- | --- |
| Code review | Correctness, missing tests, maintainability, and production-risk findings. |
| Implementation planning | Focused plans with clear module boundaries, assumptions, and validation steps. |
| System design | Architecture tradeoffs and design-pattern guidance for common production systems. |
| API documentation | Review of endpoint specs, SDK docs, integration clarity, and release readiness. |
| Commit messages | Clear scope, behavior, and validation wording. |

## Invocation

```txt
Use omniverse-engineering-skillset to review this architecture proposal.
```

The runtime behavior is defined in `skills/omniverse-engineering-skillset/SKILL.md`.
